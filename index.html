Welcome to my Github webpage..

This page is merely a repository on my github and I'm using it as a fun test of html styles and looks.

You're welcome to clone this site into your repository for a jumpstart at your own webpage. Just make sure you change the name of your repository. You'll want it to look like the following:

$yourname.github.com

Github will create a webpage for you and even offer some fun templates.

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.
